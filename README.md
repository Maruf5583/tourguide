# 🌍 Tour Guide Platform

A modern and feature-rich **Tour Guide Management Platform** built with **React + Vite + JavaScript**. This platform helps travelers discover destinations, plan smart trips, book professional guides, and manage travel experiences through an interactive and scalable web application.

---

## 📌 Project Overview

The Tour Guide Platform is a complete tourism management system designed to connect travelers with local tour guides while providing smart planning tools and administrative management features.

Users can explore destinations, create travel plans, book guides, and manage their tours. Guides can apply to become verified tour guides, manage bookings, and track their activities through a personalized dashboard. Administrators have full control over users, roles, guide approvals, and platform-wide announcements.

## ⚙️ Installation

Clone the repository

```bash
git clone <repository-link>
```

Move to project folder

```bash
cd tour-guide-project
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```
